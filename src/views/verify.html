⏳ BELUM DIVALIDASI

Surat ini masih menunggu proses validasi

Surat ini sudah terdaftar namun belum melalui proses validasi resmi.

⚠️ Surat ini belum dapat digunakan untuk keperluan resmi. Silakan hubungi petugas desa untuk proses validasi.

❌ SURAT TIDAK VALID

Surat ini tidak terdaftar atau telah dipalsukan

Surat tidak ditemukan dalam database resmi Desa Pongpongan.

⚠️ Surat ini TIDAK SAH. Jangan gunakan untuk keperluan resmi. Laporkan kepada pihak desa jika menemukan surat ini.

❌ SURAT DITOLAK

Surat ini telah ditolak dan tidak dapat digunakan

Surat ini telah melalui proses validasi dan DITOLAK oleh petugas.

⚠️ Surat ini TIDAK SAH dan TIDAK DAPAT DIGUNAKAN untuk keperluan apapun.

⚠️ GAGAL TERHUBUNG

Tidak dapat terhubung ke server verifikasi

Terjadi kesalahan saat menghubungi server.

Silakan cek koneksi internet Anda atau hubungi administrator.
